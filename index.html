1. Etiquetas iniciales y para parrafos

Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo

Es decir, las etiquetas HTML son el lenguaje utilizado para estructurar y definir el contenido en un documento HTML. Estas etiquetas se encuentran en el HTML (o Lenguaje de Marcado de Hipertexto) de cada página

1.Etiqueta HTML

Es el elemnto raiz se utiliza para dar inicio a cualquier codigo todoesto tiene que estar dentro de las etiquetas html

la etiqueta se escribe <.HTML> <./HTML>

2.Etiqueta TITLE

Titulo de la pagina El tag title (título) es un tag de código HTML que le permite darle un título a una página we

la etiqueta se escribe <.TITLE> <./TITLE>

3.Etiqueta BODY

Contenido de la pagina Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento

la etiqueta se escribe <.BODY> <./BODY>

4.Etiqueta H1 a H6

Titulos dentro del contenido Los elementos de encabezado implementan seis niveles de encabezado del documento, h1 es el más importante, y h6 , el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta

la etiqueta se escribe <. H1> <./ H1>

Asi se venlas etiquetas de H1 a H6:

Texto de prueba (H2)

Texto de prueba (H3)

Texto de prueba (H4)

Texto de prueba (H5)

Texto de prueba (H6)

5.Etiqueta FONT

Esta etiqueta se utiliza para indicar el color le la letra Indica el tamaño, color, o fuente del texto que contiene. Sus etiquetas son: (font)y (/font) (ambas obligatorias). Está definido como: Elemento desaprobado. En un doctype transicional está definido como un elemento especial y por lo tanto en línea

Asi se escribe para cambiar el color del texto que usted decee

Y su escritua en el codigo es :<.font COLOR="X" Colocando en la letra X el color que gustes ponerle a tu texto

Ahora la estructura para FONT tambien tiene como etiqueta Size la cual se etiqueta es <.size> y <./size>la cual se escribe en el codigo :<.font size="X" Colocanto en la X el numero del 1 al 7 depenfiendo que tamaño guste

Asi se ven los tamaños del 1 a 7

1=A

2=A 3=A 4=A 5=A 7=A 8=A

la etiqueta se escribe <.FONT> <.FONT>

6.Etiqueta CENTER

Se utiliza para centrar algun titulo o parrafo

la etiqueta se escribe <.CENTER> <./CENTER>

7.Etiqueta HEAD

Encabezado del documento

la etiqueta se escribe <.HEAD> <./HEAD>

8.Etiqueta P

es para los parrafos

la etiqueta se escribe <.P > <./P >

9.Etiqueta !DOCTYPE

Se utiliza para especificar la versión de HTML que se está utilizando en la página

la etiqueta se escribe <.!DOCTYPE> <./!DOCTYPE>

10.Etiqueta U

La etiqueta <.u> se utiliza para dar formato al texto subrayado en HTML. Todo lo que se escribe entre las etiquetas <.u> y <./u> aparecerá subrayado en la página web. El subrayado se utiliza para resaltar el texto, a menudo para indicar un enlace o para resaltar el contenido en una web

la etiqueta se escribe <.U> <./U>
